## Quick start


- [Download from Github](https://github.com/khalil117/khalil-touil-resume.git).
- Clone the repo: `git clone https://github.com/khalil117/khalil-touil-resume.git`.
- `npm i nextjs-material-kit`
- Install with [Bower](https://bower.io/?ref=creativetim): ```bower install nextjs-material-kit```.
